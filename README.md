# AI Ethics Assignment

This repository contains the submission for the AI Ethics assignment themed "Designing Responsible and Fair AI Systems".  

## Contents:
- Part 1: Theoretical Understanding (PDF)
- Part 2: Case Study Analysis (PDF)
- Part 3: Bias Audit of COMPAS Dataset (Jupyter Notebook + PDF report)
- Part 4: Ethical Reflection (PDF)
- Bonus Task: Ethical AI Policy in Healthcare (PDF)
- Code folder: Python scripts and requirements
- Visuals folder: Charts and graphs from audit

## Instructions:
1. Install dependencies: `pip install -r code/requirements.txt`
2. Run `Part3_Bias_Audit.ipynb` or `Part3_Bias_Audit.py` to reproduce the bias audit and visualizations.
